Panduan Tata Cara Admin
1. Manajemen Akun
Lihat daftar akun di menu Akun.
Tambah akun baru dengan klik Buat Akun Baru.
Edit akun dengan tombol Edit.
Hapus akun dengan tombol Hapus lalu konfirmasi.
2. Manajemen Dokumen
Masuk ke menu Dokumen.
Admin dapat melihat daftar seluruh dokumen dari semua pengguna.
Cek status laporan (sudah diunggah / belum diunggah).
Periksa batas waktu unggah laporan yang tertera di daftar.
Klik Detail untuk melihat informasi lengkap dokumen.
Gunakan tombol Unduh Surat atau Unduh Laporan untuk mengakses file.
Hapus dokumen/laporan dengan tombol Hapus bila diperlukan.
Gunakan fitur Pencarian (Search) untuk menemukan dokumen berdasarkan nomor surat, nama tim, atau irban.
3. Logout
Klik tombol Logout di navbar untuk keluar.
