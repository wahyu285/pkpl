Tata Cara Penggunaan Aplikasi REGAI
  1. Login dengan username dan password Anda.
  2. Masuk ke menu Unggah Dokumen untuk menambahkan surat tugas atau laporan.
  3. Isi nomor surat, tanggal, nama tim, jabatan, dan unggah file PDF.
  4. Gunakan menu Lihat Dokumen untuk melihat daftar dokumen yang sudah diunggah.
  5. Klik tombol Unggah Laporan pada dokumen yang belum memiliki laporan.
  6. Gunakan menu Profil untuk mengubah data akun Anda.
  7. Jangan lupa klik Logout setelah selesai menggunakan aplikasi.
